# Railway
# 🚆 C++ Program – Railway Reservation System

This project is a simple **Railway Reservation System** written in C++.

It uses:

- Class & Object  
- Constructors (Default, Parameterized, Copy)  
- Destructor  
- Static Member  
- Vector (STL)  
- Menu-driven system  

The system can perform:

- Add Train  
- Display All Trains  
- Search Train by Number
